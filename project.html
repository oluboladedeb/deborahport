DEVELOPMENT CYCLE

My projects are implemented using software development lifecycle approach to ensure effectiveness
and functionality of websites. I strategize, design, code, test and deploy the deliverables.

Strategic planning is the heart of software project development lifecycle. Plans are documented, project deliverables requirement are defined and project schedule is outlined.

Following project requirements, web architecture and layout are defined. Wireframing tools are employed within this phase.

Once the design phase is over, tasks are divided into units and code is implemented following defined design concept.

Project is deployed in the testing environment and the entire functionality of the site is tested. This continues until software is bug-free, stable and functional. Finally, project goes live when all requirement test are satisfied.
